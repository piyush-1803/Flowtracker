# 🌊 FlowTracker

A modern, Fluent-inspired desktop automatic productivity monitoring and active workspace time tracker catalogued dynamically by workflows.

---

## ✨ Features

* **Automatic Activity Monitoring**
* **Dynamic Workflow Cataloging**
* **Fluent-Inspired UI**
* **AI-Powered Insights**

---

## 🛠 Tech Stack

* **Core:** HTML5, CSS3, TypeScript, Vite
* **Backend:** Firebase
* **AI:** Google AI Studio Framework

---

## 🚀 Installation & Setup

1. **Clone the repo:** `git clone https://github.com`
2. **Navigate:** `cd Flowtracker`
3. **Install:** `npm install`
4. **Configure:** Create `.env.local` based on `.env.example` and add `GEMINI_API_KEY`.
5. **Run:** `npm run dev`
