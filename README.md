> A modern, Fluent-inspired desktop automatic productivity monitoring and active workspace time tracker catalogued dynamically by workflows.

FlowTracker is built to sit silently in your workspace background, intelligently observing your active tasks and organizing your time metrics dynamically by specific workflows—eliminating the need to manually punch stopwatches or segment your day.

---

## ✨ Features

* **Automatic Activity Monitoring:** Tracks active workspace time seamlessly without disrupting your creative or technical flow.
* **Dynamic Workflow Cataloging:** Automatically groups and categorizes your productivity analytics based on active development or task environments.
* **Fluent-Inspired UI:** Features a sleek, modern desktop interface built with Microsoft's Fluent Design principles in mind.
* **AI-Powered Insights:** Integrated via Google AI Studio templates to provide structural and context-aware workspace optimization feedback.

---

## 🛠️ Tech Stack

* **Frontend Core:** HTML5, CSS3, [TypeScript](https://www.typescriptlang.org/)
* **Build Tool:** [Vite](https://vitejs.dev/)
* **Backend/Applet Management:** Firebase (`firebase-applet-config.json`)
* **Environment Engine:** Google AI Studio Framework

## 🚀 Getting Started

### Prerequisites

Ensure you have [Node.js](https://nodejs.org/) installed on your machine (LTS version recommended).

### Local Installation

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/piyush-1803/Flowtracker.git](https://github.com/piyush-1803/Flowtracker.git)
   cd Flowtracker
Install dependencies:

Bash
npm install
Configure Environment Variables:
Duplicate the provided example environment file and add your Gemini API key.

Bash
cp .env.example .env.local
Open .env.local and configure your API key:

Code snippet
GEMINI_API_KEY=your_actual_gemini_api_key_here
Run the Development Server:

Bash
npm run dev
The application will boot up locally. Follow the local address output in your terminal (usually http://localhost:5173).
